🧪 Teste Manual - Validação de Compra

Objetivo: Verificar se o sistema impede salvar compras sem produtos válidos.

📋 Preparação

  1. Abra o navegador e vá para: http://localhost:5173/compras/nova
  2. Abra o Console do Desenvolvedor (F12 → Console)
  3. Certifique-se de que o servidor frontend está rodando

🔍 Teste 1: Tentar Salvar Compra Completamente Vazia

  1. Não preencha nenhum campo
  2. Clique no botão "Salvar"
  3. Resultado Esperado:
    • ❌ Deve aparecer erro: "Obra é obrigatória"
    • ❌ Deve aparecer erro: "Data da compra é obrigatória"
    • ❌ Deve aparecer erro sobre itens válidos
    • 🚫 NÃO deve salvar a compra

🔍 Teste 2: Preencher Campos Obrigatórios, Mas Deixar Itens Vazios

  1. Selecione uma Obra
  2. Preencha a Data da Compra
  3. Deixe a tabela de itens vazia (ou com o item inicial vazio)
  4. Clique no botão "Salvar"
  5. Resultado Esperado:
    • ❌ Deve aparecer erro: "Uma compra deve ter pelo menos um item válido com material, quantidade e valor unitário preenchidos."
    • 🚫 NÃO deve salvar a compra
    • 📝 No console deve aparecer logs com 🔍 VALIDAÇÃO

🔍 Teste 3: Preencher Item Parcialmente

  1. Mantenha Obra e Data preenchidas
  2. Na tabela de itens, preencha apenas o Material
  3. Deixe Quantidade e Valor Unitário vazios
  4. Clique no botão "Salvar"
  5. Resultado Esperado:
    • ❌ Deve aparecer erros específicos nos campos vazios
    • 🚫 NÃO deve salvar a compra

✅ Teste 4: Preencher Corretamente (Deve Funcionar)

  1. Mantenha Obra e Data preenchidas
  2. Na tabela de itens:
    • Selecione um Material
    • Preencha Quantidade (ex: 10)
    • Preencha Valor Unitário (ex: 5.50)
  3. Clique no botão "Salvar"
  4. Resultado Esperado:
    • ✅ Deve salvar a compra com sucesso
    • 📝 No console deve aparecer logs com 🚀 SUBMIT

🚨 O Que Observar no Console

Durante os testes, observe no console do navegador:

  • 🔍 VALIDAÇÃO: Iniciando validateForm com items: - Mostra os itens sendo validados
  • 🔍 VALIDAÇÃO: Nenhum item válido encontrado! - Quando não há itens válidos
  • 🚀 SUBMIT: VALIDAÇÃO FALHOU - Quando a validação impede o envio
  • 🚀 SUBMIT: Enviando payload: - Quando a validação passa e envia os dados

  1. Copie os logs do console
  2. Verifique se há mensagens de erro na tela
  3. Anote o comportamento observado vs esperado
